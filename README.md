# Yellow Temperance Addon

这是一个为Ripples of the Past Minecraft模组制作的替身扩展模组，添加了黄色节制替身的能力。

This is a Stand addon for Ripples of the Past Minecraft mod that adds the Yellow Temperance Stand abilities.

## 功能特性 / Features

- **黄色节制替身** - 基于白蛇伪装系统的核心能力
  - **Yellow Temperance Stand** - Core abilities based on Whitesnake's disguise system
- **高级冒充系统** - 增强的伪装机制，支持玩家皮肤
  - **Advanced Impersonation** - Enhanced disguise mechanics with player skin support
- **侵蚀效果** - 使用修改的青春岁月霉菌层创建的自定义状态效果
  - **Erosion Effects** - Custom status effects using modified Green Day mold layer
- **动态渲染** - 自定义伪装层和视觉效果
  - **Dynamic Rendering** - Custom disguise layers and visual effects
- **玩家人偶系统** - 高级玩家表示和渲染系统
  - **Player Doll System** - Advanced player representation and rendering
